## Cheatsheet

Remember that upper case `C` refers to the <kbd>CTRL</kbd> key and lowercase `c` refers to the <kbd>c</kbd> key. E.g. `C-c` would mean _hold down_ <kbd>CTRL</kbd> _and press the_ <kbd>c</kbd> _key_

| command     | action     |
| ----------- | ---------- |
| `C-c` `C-x` | Exit `emacs` |
